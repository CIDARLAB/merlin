Merlin

M.A.G.E Optimization tool

Developed by CIDAR BU

  • Given an list of Desired Oligos, Merlin aims to generate oligos for MAGE experiments to optimize genome-to-oligo homology and oligo-to-oligo homology.

    Getting Started


    Merlin takes the following inputs

  • Initial Genome FASTA File sample

  • Input Parameter File: sample

  • Input Target File: sample

    • Please note that target/param files are tab delimited


  • Upload your starting Genome File

  • Upload the parameter and target files

  • Click MAGE to run mage

  • Click MERLIN to run merlin
